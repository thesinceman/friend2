# friend2
a small python project that works in the linux terminal

# instalation

# arch

install python3 (if you dont already have it)
```
sudo pacman -S python3
```
install git (if you dont already have it)
```
sudo pacman -S git
```
(optinal) update your system
```
sudo pacman -Syu
```
clone the repository 
```
git clone https://github.com/thesinceman/friend2
```
cd into the repository
```
cd friend2
```
run the program
```
python3 friend2.py
```

# debian

install python3 (if you dont already have it)
```
sudo apt install python3
```
install git (if you dont already have it)
```
sudo apt install git
```
clone the repository
```
git clone https://github.com/thesinceman/friend2
```
cd into the repository 
```
cd friend2
```
run the program
```
python3 friend2.py
```
